##### 변수 위치에 따른 구분

- 멤버변수 (fileld)
  - 메서드 밖에 선언
  - static이 붙은 변수: 클래스 변수
  - static이 붙지 않은 변수: 인스턴스 변수

- 지역변수( local)
  - 메서드 안에 선언

  - 지역변수는 메서드 내에서만 사용 가능
